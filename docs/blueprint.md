# **App Name**: Data Insights Portfolio

## Core Features:

- Dynamic Role Display: Showcase current role ('Business Intelligence Analyst') and transition to other roles ('AI Automation Expert', 'Marketing Analyst', 'Financial Analyst', 'Data Analyst', 'Operations Manager') with a typing animation effect.
- GitHub Project Fetch: Automatically fetch and display projects from the specified GitHub repository.
- Intelligent Content Generation: Automatically generate summaries and taglines for portfolio projects, pulling information from project descriptions and README files. Use a tool to choose relevant information for the summarization.
- Resume Integration: Display work experience, education, and certifications from the resume PDF in structured sections.
- Contact Form Submission: Enable users to submit messages via a contact form, processed through Firebase Functions.
- Responsive Layout: Ensure the portfolio is fully responsive across desktop, tablet, and mobile devices.
- Downloadable Resume: Provide a prominent button for users to download the resume.

## Style Guidelines:

- Primary color: Deep charcoal (#333333) for a professional and modern base. Inspired by the desire for a sophisticated design, reflecting the professionalism of a Business Intelligence Analyst.
- Background color: Light gray (#F0F0F0), providing a subtle backdrop to ensure readability and focus.
- Accent color: Crimson (#DC143C), used for highlights, interactive elements, and calls to action to add energy and draw attention.
- Body and headline font: 'Inter', a sans-serif font for clean and modern readability.
- Use simple, line-based icons to represent skills and technologies, ensuring they are consistent and minimalist.
- Implement a minimalist layout with ample white space, focusing on clear sections for easy navigation and content consumption.
- Incorporate subtle, smooth transitions and animations for an elegant and engaging user experience, such as fading in sections on scroll.
